# Stock Portfolio Tracker

## Description
This project is a simple Python-based Stock Portfolio Tracker that calculates the total value of a user's stock investments. Users enter the stock names and quantities they own, and the program computes the total investment value using predefined stock prices. The final result is also saved to a text file.

## Features
- Accepts user input for stock names and quantities.
- Uses predefined stock prices.
- Calculates the total portfolio value.
- Saves the result to a `portfolio.txt` file.
- Simple and beginner-friendly Python program.

## Technologies Used
- Python 3

## How to Run
1. Download or clone this repository.
2. Open the project in VS Code or any Python IDE.
3. Run the program:
   ```bash
   python stock_portfolio.py
   ```
4. Enter the stock names and quantities when prompted.
